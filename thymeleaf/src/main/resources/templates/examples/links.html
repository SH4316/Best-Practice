CDN 리소스 처리

Best Practice ✅
로컬 리소스가 있으면 사용하고, 없으면 CDN을 사용합니다.
Bad Practice ❌
인터넷 연결이 없으면 리소스를 로드할 수 없습니다.
Best Practice 요약
  • 항상 URL 표현식(@{...})을 사용하여 동적 URL을 생성하세요
  • 정적 리소스도 URL 표현식을 사용하여 일관성을 유지하세요
  • 쿼리 파라미터는 URL 표현식 내에서 처리하세요
  • 링크 상태를 처리하여 현재 페이지를 시각적으로 표시하세요
  • CDN 리소스는 로컬 리소스와 함께 사용하여 안정성을 높이세요
